About the BFSI Security Hub

This application serves as a demonstration of integrating specialized machine learning models into a modern web environment.

Core Components

  • **Fraud Detection:** Custom binary classification model (`fraud_detector.pkl`).
  • **Chatbot:** Fine-tuned LLM (`phi3-bfsi-finetuned`) specialized for financial, banking, and insurance (BFSI) queries.
  • **Backend:** Python Flask API for model serving and data management.
  • **Frontend:** Single-page application using HTML, JavaScript, and Tailwind CSS for a responsive, multi-page experience.
